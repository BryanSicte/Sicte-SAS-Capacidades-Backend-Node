# 📱 Sicte SAS CCOT Backend



## Ejecutar proyecto Primera Vez
```bash
git clone https://github.com/BryanSicte/Sicte-SAS-Capacidades-Backend-Node.git
cd "Sicte-SAS-CCOT-v2"
npm install --legacy-peer-deps
npm run start:dev
```




## Instalación local
### Instalar node_modules
```bash
npm install
```
### Ejecutar el servidor
```bash
npm start
```
### Ejecutar el proyecto si se tiene modemon en el package.json
```bash
npm run dev
```
